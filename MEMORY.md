# MEMORY.md - Your Long-Term Memory

## 사주 프로그램 개발 프로젝트 (2026-04-02 시작)
- **목표:** Windows/Mac 로컬 구동 가능한 사주 풀이 프로그램 구현.
- **핵심 기능:**
  - 1단계: 사주원국, 오행, 십성, 신강신약, 대운수/표 계산.
  - 2단계: 애정운, 재물운, 이성운, 직업운, 건강운, 궁합 등 항목별 텍스트 풀이.
- **철학:** 단점을 장점이나 가능성으로 해석하는 긍정적인 방향, 이해하기 쉬운 물상 표현, 흥미로운 문체 지향.
- **2026-04-03 업데이트:** 
  - 프로젝트 구조 모듈화 (`engine/`, `ui/`, `data/`) 완료.
  - 현대적 물상론 기반 해석 엔진 (`interpreter.py`) 개발 착수.
  - PySide6 기반 GUI (`main_window.py`) 기본 프레임워크 구현.
  - '희망 지향적' 해석 철학 적용 (무재 사주 재해석 등).

  - **'인터랙티브 클라우드 포탈(Interactive Cloud Portal)' v3.0 Early Access:** 
    - **인터랙티브 포탈 제작:** 사용자가 이름/생년월일시를 입력하고 'RUN ENGINE' 버튼을 눌러 엔진이 구동되는 과정을 시각적으로 확인(터미널 로그 애니메이션)할 수 있는 `X-SAJU_CLOUD_INTERACTIVE_PORTAL.html` 제작.
    - **디자인 가이드라인 반영:** '사(사)', '병(병)' 같은 중복 표기를 전면 제거하고, 모든 지표를 **한문(한글)** 형식으로 정형화.
    - **초슬림 한문 폰트:** 원국(8자)에 사용되는 얇은 폰트(Weight 100)를 리포트 전체 한문에 적용하여 가독성과 전문성 극대화.
    - **정체성 확립:** 단순히 파일을 전달하는 것을 넘어, 사용자가 직접 **'작동하는 소프트웨어'**를 경험할 수 있도록 웹 기술을 활용한 인터랙티브 환경 제공.
    - **분석 깊이 완성:** 120개 이상의 분석 카테고리와 2026년 적월(Red Moon) 특화 로직을 결합한 **183페이지 분량의 마스터 리포트** 생성 및 배포 성공.
    - **개인 맞춤형 최적화:** 주인님(壬戌 일주)의 실제 생년월일시(1988-04-27 01:04 Solar) 데이터를 반영하여 고밀도 에디토리얼급 텍스트 구현.
    - **디자인 가이드(v0.8.8) 전면 적용:** 얇은 고딕 한문과 미니멀한 타이포그래피를 183페이지 전면에 일관되게 이식함.
    - **배포 전략:** HTML5 기반의 초고해상도 배포 시스템을 통해 모바일/PC 범용성 및 PDF 출력 품질을 동시에 확보함.
    - **분석 엔진 고도화 목표:** 
        1. 60갑자(Il-ju)별 상세 물상론 데이터셋 구축.
        2. 용희신(Lucky Elements) 정밀 판별 로직 구현.
        3. 12운성 및 신살(Gods/Demons) 통합 분석 시스템 설계.
        4. 대운/세운 결합을 통한 '월간/일간 정밀 로드맵' 생성 기능 강화.
    - **디자인 보류:** 현재까지 확정된 v0.8.5 가독성 가이드를 모든 페이지의 '표준'으로 삼고, 추가적인 디자인 수정은 엔진 개발 완료 후 진행하기로 함.
  - **'가독성 최적화(Legibility Optimization)' v0.8.3:** 주인님의 스크린샷 피드백(5a33b... / 453a0...)을 바탕으로 디자인 시스템을 최종 튜닝함.
    - **색상 비율:** 시/일/월/연 및 십성은 'Black 10%' (밝은 회색 #E6E6E6), 보조 한글 및 지표 레이블은 'Black 30%' (차분한 회색 #B3B3B3)로 설정하여 명확한 위계 확보.
    - **폰트 확정:** 첫 번째 시안의 섬세한 Serif 한자 폰트를 고정하고 크기를 82px로 상향.
    - **레이아웃:** 전체 중앙 정렬 및 작은 글자 크기(1~3px 상향) 조정을 통해 모바일과 PC 모두에서 최상의 가독성 구현.
  - **외부 원격 접속(Tailscale) 도입 제안:** 집 밖에서도 앱으로 완벽한 제어를 원할 경우, 보안 터널인 테일스케일(Tailscale) 설치를 통한 하이엔드 원격 환경 구축이 필요함을 안내함.
  - **무채색의 세밀한 계조(Grayscale Gradation) 활용:** 단순히 흑백이 아닌, #111부터 #555까지의 세밀한 회색 계조를 사용하여 강조할 부분만 은은하게 드러내는 고난도 타이포그래피 기법을 적용함.
  - **시각적 위계(Visual Hierarchy)의 예술적 재정의:** 제목과 핵심 강조어는 가장 굵게(900), 사주 원국과 본문은 가장 얇게(100) 배치하여 시선이 자연스럽게 흐르면서도 지루하지 않은 '보는 재미'가 있는 레이아웃을 구축함.
  - **리포트 뷰어 UX 개선:** 가로 스크롤 방식에서 **세로 스크롤(Vertical Scroll)** 방식으로 레이아웃 전면 수정. A4 비율의 페이지를 위아래로 휠을 돌려가며 편하게 읽을 수 있도록 애플 스타일의 부드러운 배경색과 그림자 효과 적용.
